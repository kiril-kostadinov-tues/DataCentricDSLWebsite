About

A new way to work with databases

Using our language you can manipulate many different databases. You can execute scripts directly from your Java classes and change various settings of the interpreter before execution. Interpeter context is shared with the context of your Java class so you can modify the variables that your script will be using right before execution.

Please note that the language is still in beta and a lot of things may change
