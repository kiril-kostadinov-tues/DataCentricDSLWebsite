Using query

The keyword 'query' is used to query a database and get the result. You can control which database is queried using the variable 'dataSource'.

Query expects a query string. It can be provided as a string literal.

query ''SELECT * FROM Students'';

The query string can also be provided using a variable or an expression.

a = ''SELECT * FROM Students'';
query a;
query ''SELECT * '' + ''FROM Students'';

You can assign the result of the query to a variable.

result = query ''SELECT * FROM Students'';
